# Competitor Analyzer

**Competitor Analyzer** — это учебный проект для анализа конкурентов на маркетплейсах (Wildberries, Ozon и др.).  
Приложение собирает данные по товарам конкурентов и помогает формировать карточку товара.

## Возможности

- Поиск и анализ карточек конкурентов по параметрам
- Парсинг данных с маркетплейсов (через API и/или Selenium)
- Обработка текстовой информации и изображений
- Формирование черновика карточки товара
- Графический интерфейс для работы с результатами

## Структура проекта

- `main.py` — точка входа в приложение
- `run.py` — запуск GUI‑версии
- `gui/` — графический интерфейс (окна, форматирование, работа с API)
- `services/` — бизнес‑логика и сервисы (анализ конкурентов, парсинг, история)
- `models/` — модели запросов и ответов
- `examples/` — примеры результатов анализа
- `selenium_scraper.py` — парсер на базе Selenium
- `wildberries_api.py` — работа с API Wildberries
- `config.py` — загрузка настроек и переменных окружения
- `requirements.txt` — список зависимостей Python

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/shkapao28-ship-it/competitor-analyzer.git
   cd competitor-analyzer
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Создайте файл `.env` в корне проекта и добавьте туда свои ключи и настройки (API‑ключи, токены и т.п.):

   ```env
   API_KEY=ваш_ключ
   # другие переменные окружения
   ```

5. Запуск приложения:
   ```bash
   python main.py
   ```
   или для GUI‑версии:
   ```bash
   python run.py
   ```

## Заметки

- Файл `.env`, виртуальное окружение `.venv`, артефакты сборки и результаты работы добавлены в `.gitignore` и не попадают в репозиторий.
- Проект выполнен в рамках курса по промт‑инжинирингу и используется для обучения работе с LLM, API и автоматизацией анализа данных.
